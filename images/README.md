# Images Folder

Add your images here for the Exercise Coach Charts application.

## Recommended Images

### Required for Current Implementation:
- **logo.png** - Main Exercise Coach logo (recommended: 80x80px or similar aspect ratio)
  - Will appear in the header of the clients page
  - Should have a white or transparent background

### Optional Images (For Future Enhancement):
- **hero-image.png** - Large background or hero image for login page
- **trainer-photo.jpg** - Sample trainer/coach image
- **studio-photo.jpg** - Studio or gym environment photo
- **icon-*.png** - Small icons for different sections

## Image Guidelines:
- Keep file sizes optimized (compress before adding)
- Use PNG for logos and icons (supports transparency)
- Use JPG for photos and larger images
- Recommended dimensions for logo: 80-120px width
